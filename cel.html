cell paradigm

Following sentences define the paradigm of (i/o) cells.
Each cell ..
  • .. is independent
  • .. is simple
  • .. is stupid
  • .. doesn't know that other cells exists
  • .. has no side effects
  • .. has its own memory
  • .. has 0 to N inputs
  • .. has 0 to 1 outputs
  • .. calc output from inputs
  • .. runs parallel but concurrent to other cells
  • .. has 0 to N in connections to other cells
  • .. has 0 to M out connections to other cells
